symbest

This program searches a large set of particles for those with the best symmetry.

symbest <input file> <output file> sym=<Cn> [keep=<N>] [mirror=<mirror out>] [mask=<rad>] [imask=<rad>]

Parameters:


<input file>Input file constining raw particles
<output file>Output file with the selected best particles
sym=<Cn>Symmetry to search for
[keep=<N>]Number of particles to keep
[mirror=<mirror out>]Also search for particles with mirror symmetry
[mask=<rad>]Mask radius
[imask=<rad>]Inside mask radius

Usage:

symbest start.img best.img sym=c4

Description

This program searches a set of particles for those with the best rotational and/or mirror symmetry. They should be precentered with cenalignint or a similar utility. This program is generally indirectly through the use of startcsym or starticos.
