circleboxer

This program is used for offline boxing of micrographs using circular contrast regions

circleboxer input=<filespec> ptclsize=<diameter> [boxsize=<size>] [output=<filespec>] [dbout=<filespec>]

Parameters:


input=<filespec>Image to be boxed (MRC format only!!!)
ptclsize=<diameter>Max diameter of the particles in pixels
[boxsize=<size>] Final box size in pixels (edge length)
[output=<filespec>]Output file for boxed images
[dbout=<filespec>]Output box database file

Usage:

circleboxer input=jj1232.mrc ptclsize=250 boxsize=274 output=jj1232.hed dbout=jj1232.box

Description

This program performs offline boxing of micrographs by comparing the density inside a circle with the density immediately outside the circle, and using the standard deviation just outside the particle as an additional exclusion criteria
