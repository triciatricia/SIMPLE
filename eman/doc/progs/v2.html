V2

 

An interactive 2d, 3d images viewing program


Usage
v2 [image1] ... ...
 

Typical Usage
v2 thread.1a.img thread.2a.mrc ...

Description
This is a simple program designed to interactively display 2d or 3d images side by side.If the images file include multiple images , this program will use image matrix to display it.
