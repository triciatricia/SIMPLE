classalignall

This will align the images in all of the cls*img files in the current directory

classalignall <iterations> [nobad] []

Parameters:


Description

This program will align the images in each cls*img file to generate a class average. It is simply a wrapper for classalign2. All of the options are documented there. The <input file> parameter is excluded in this case, since this program automatically operates on all cls files.
