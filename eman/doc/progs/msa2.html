msa2

This performs multivariate statistical analysis on 2 sets of invariants

msa2 <input fp 1> <input fp 2> <out foot file 1> <out foot file 2> <final n2 value>

Parameters:


<input fp 1>A footprint file to be analyzed
<input fp 2>A footprint file to be analyzed
<out foot file 1>The reduced dimensionality output file
<out foot file 2>The reduced dimensionality output file
<final n2 value>Dimensionality of the output

Usage:

msa file.fp file.msa.fp 8

Description

This program will perform multivariate statistical analysis on input 1 (produced by one of the footprint programs. It then uses the new basis set to reduce the dimensionality of input 2. The output is two footprint files with reduced dimensionality.
