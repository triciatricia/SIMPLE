iv2dejavu

This program converts helices in Open Inventor format to DejaVu secondary structure elements file format

iv2dejavu <input IV file> <output DejaVu file> [verbose]

Parameters:


<input IV file>The Open Inventor format file
<output DejaVu file>The output DejaVu format file
[verbose]will output some intermediate information

Usage:

iv2dejavu in.iv out.dejavu verbose

Description

This program uses Open Inventor library, so it is only working on SGI at this moment.

This program works correctly only for the special Open Inventor file containing only cylinders (helices).

Visit web page Helixhunter and Foldhunter for more information.
