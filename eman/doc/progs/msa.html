msa

This performs multivariate statistical analysis on a set of invariants

msa <input> <out foot file> <final n2 value>

Parameters:


<input>A footprint file to be analyzed
<out foot file>The reduced dimensionality output file
<final n2 value>Dimensionality of the output

Usage:

msa file.fp file.msa.fp 8

Description

This program will perform multivariate statistical analysis on a footprint file (produced by one of the footprint programs. The output is the same footprint file with reduced dimensionality.
