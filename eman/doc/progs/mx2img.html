mx2img

This program converts a distance matrix to an MRC image

mx2img <matrix file> <MRC file>

Parameters:


<matrix file>Input matrix file
<MRC file>Output MRC image

Usage:

mx2img matrix.dat matrix.mrc

Description

This program will convert the matrix file created by the matrix option of classesbymra into a standard MRC image. This is generally used in conjunction with glmatrix or matrixembed.
