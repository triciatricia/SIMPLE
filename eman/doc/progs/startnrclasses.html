startnrclasses

This program takes a set of particle images and generates reference free class averages

startnrclasses <infile> <# pt per class> [imask=<rad>] [mask=<rad>] [parm=<# fp>] [proc=<n proc>] [nobad]

Parameters:


<infile>Input file of raw (centered) particles
<# pt per class>Approximate number of particles to use for each class average
[imask=<rad>]Inside mask radius for data exclusion
[mask=<rad>]Mask radius
[parm=<# fp>]Number of MSA basis vectors to use in classification, default is usually adequate
[proc=<n proc>]Processors to run on
[nobad]Exclude empty classes from the result, losing correlation to cls files

Usage:

startnrclasses start.img 50

Description

This program is used to generate reference free class averages. It is fully automated. All you specify is the approximate number of particles to use for each class. Generally this routine will produce at least a few good representative class averages. It may not cover all possible orientations, and there may be many class averages containing almost identical averages. This program is generally used in conjunction with startAny. First startnrclasses is used to generate a large number of class averages. The user then manually selects several representing as diverse a range of orientations as possible, and runs them through startAny to generate an initial model for refinement.
