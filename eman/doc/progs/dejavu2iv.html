dejavu2iv

This program converts helices in DejaVu secondary structure elements file format to Open Inventor format

dejavu2iv <input DejaVu file> <output IV file> [verbose]

Parameters:


<input DejaVu file>The input DejaVu format file
<output IV file>The output Open Inventor format file
[verbose]will output some intermediate information

Usage:

dejavu2iv in.dejavu out.iv verbose

Description

Visit web page Helixhunter and Foldhunter for more information.
