eulerclassrefine

This program will try to refine the Euler angles for a set of class-averages using the concept of common-lines

eulerclassrefine <classfile>

Parameters:


Description

This program will attempt to refine the Euler angles for a set of class-averages. It relies on the images having relatively low noise levels. It also relies on the presence of moderately accurate Euler angles already being present in the image headers. It will strive to locally adjust Eulers for more self-consistency.
