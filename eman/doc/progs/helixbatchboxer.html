helixbatchboxer

This program is used for offline helixboxing of micrographs

helixbatchboxer [input=<filespec>] [output=<filespec>] [length=<length>] lengt [width=<width>] widt [dbbox=<filespec>] [nointerp] No interpolatio [sep] Output boxed out particle in seperate file

Parameters:


[input=<filespec>]Image to be boxed (MRC format only!!!)
[output=<filespec>]Output file for boxed images
[length=<length>] lengt[length=<length>] length
[width=<width>] widt[width=<width>] width
[dbbox=<filespec>]Input box database
[nointerp] No interpolatio[nointerp] No interpolation
[sep] Output boxed out particle in seperate file[sep] Output boxed out particle in seperate files

Usage:

helixbatchboxer dbbox=jj0441.box input=jj0441.mrc output=0441.hed

Description

This program does offline boxing of particles using a box database or an automatic boxing routine. Please note that it will work only on MRC images.
