cenalignacf

This program will center a set of particles using an autocorrelation function

cenalignacf <input file> [mask=<rad>] [maxshift=<pix>]

Parameters:


<input file>File containing images to be centered
[mask=<rad>]Circular mask radius in pixels
[maxshift=<pix>]Maximum shift to use in centering

Usage:

cenalignacf start.hed mask=42 maxshift=8

Description

This program centers a set of particles individually using an autocorrelation function. Clearly this works well only if the particles have a suitable shape in all orienations.
