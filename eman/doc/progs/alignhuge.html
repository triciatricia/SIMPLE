alignhuge

Determines the rotation and transformation between images in a focal pair

alignhuge <image to align> <fixed reference> <output> [mrc8bit] [maxscale=<float>]

Parameters:


<image to align>First input (MRC only!)
<fixed reference>Second input (MRC only!)
<output><input 1> aligned to <input 2>
[mrc8bit]Output 8-bit MRC file to conserve space
[maxscale=<float>]Maximal fraction of scale difference. default to 0.05

Usage:

alignhuge im0221.mrc im0222.mrc

Description

This will determine the relative alignment parameters between two very large images. It's designed for use with 'identical' images (except for defocus), ie - a focal series. It will output the aligned image and display the rotation/translation parameters. The file "alignhuge.txt" will contain the image to align, reference image, x, y, rotation, and scale.

A compile-time option exists to determine scaling differences as well as rot/trans. The program must be recompiled to disable this behavior. On JEOL microscopes, the scaling error is below the error limit for this program.
