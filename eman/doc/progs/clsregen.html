clsregen

This program uses particle.log to regenerate cls files for a particular iteration.

clsregen <iteration>

Parameters:


<iteration>The 'refine' iteration number to extract.

Usage:

clsregen 1

Description

This program will take the 'particle.log' file, and extract information on the particle classification for a particular iteration of the refine command and regenerate 'cls' files. This can be useful if you wish to run a t-test on a previously completed iteration.

Note: This will NOT regenrate the appropriate projections, though it will put in references to proj.hed which will work if it IS generated appropriately. That is, you need to rerun proc3d on the appropriate threed.?a.mrc file.
