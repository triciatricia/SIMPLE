cloud

This program performs MSA based filtering of a set of particles

cloud <input> <output> <elements> [1 writes eigenims.img]

Parameters:


Description

This program is classified as experimental and is not documented yet. This program will take a set of 2d aligned particles and perform MSA analysis on them. The resulting MSA basis vectors (images) will then be used as a new basis for the raw images. This is equivalent to a filtering operation.
