qsegment

A graphical program for segmentation and simple exploration of 3D models


Usage
qsegment

Description

This program is used to interactively segment (split into pieces) a 3D volume dataset. It can also be used to examine 3D data slice by slice from any angle. This is the initial release of this program, and the documentation is not complete yet. email sludtke@bcm.tmc.edu if you need assistance.
