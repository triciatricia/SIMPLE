ctfavg2

Averages a set of 2D images with ctf correction

ctfavg2 <infile> <outfile> [res=<in A>] [ctfcw=<file>] [ctfc=<lp res>] [keepsig=<sig>] [keepval=<thr>] [mask=<rad>] [fileali] [refali] [copyeuler] [refmask] [ref0] [copyrefs] [even|odd] [snrmult=<val>] [nolog] [comment=<str>] Sets the comment string in the output file (if supported

Parameters:


<infile>Set of images to be averaged
<outfile>Output, 1 image will be appended
[res=<in A>]Resolution in A if structure factor is not used
[ctfcw=<file>]Structure factor file used to fit data in infile
[ctfc=<lp res>]Lowpass filter resolution for ctf correction without wiener filter
[keepsig=<sig>]Sigma multiplier for discarding particles
[keepval=<thr>]Fixed threshold for discarding particles
[mask=<rad>]Circular mask
[fileali]Use alignments from file
[refali]Do a single pass 2D alignment of each particle to the reference (ref0)
[copyeuler]Used only if the filspec is cls####, will read the Eulers from proj.hed
[refmask]Only for cls#### files, will mask based on proj.hed
[ref0]First image is a reference image
[copyrefs]Copy proj.hed image into the output file
[even|odd]Uses only even or odd numbered particles
[snrmult=<val>]Multiply the SNR by this number when Wiener filtering
[nolog]Do not log a record of this program in .emanlog
[comment=<str>] Sets the comment string in the output file (if supported[comment=<str>] Sets the comment string in the output file (if supported)

Usage:

ctfavg2 cls0000.lst classes.hed ctfcw=groel.xray

Description

This program will average a set of prealigned images with CTF correction.
