footprint3

This program generates a set of rotational and translational invariants for use in classification

footprint3 <input file> <output file> [euler]

Parameters:


<input file>Input file with image set to be 'footprinted'
<output file>Output file for binary footprints
[euler]Also writes euler angles to binary footprint file

Usage:

footprint start.img start.fp euler

Description

This program determines a set of rotational and translational invariants for a set of images and writes them to a binary file. There are several different footprint programs. Each one uses a different technique for generating invariants. All of these programs are classified as experimental, and are not used in the normal refinement procedure.
