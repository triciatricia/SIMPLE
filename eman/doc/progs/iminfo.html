iminfo

This program provides basic information about image files

iminfo <input file> [all] [stat] [first=<n>] [last=<n>]

Parameters:


<input file>File to examine
[all]Give detailed information about each image in a stack file
[stat]Statistical information about the image(s)
[first=<n>]Start with particle number n.
[last=<n>]End with particle number n.

Usage:

iminfo start.img

iminfo proj.img all

Description

This is a simple program which provides information on any image file supported by EMAN. The 'all' option gives detailed info on each image in a stacked image file.
