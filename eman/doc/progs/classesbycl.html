classesbycl

This classifies a set of centered particles using self common lines.

classesbycl <particles> <refrences> [assigneuler] [eulerinref] [frac=<num>/<denom>] [verbose[=<n>]] [matrix] [logit=<label>]

Parameters:


<particles>Raw particle filespec (must have a .cl. version)
<refrences>Reference image filespec (must have a .cl. version)
[assigneuler]Assigns Eulers to each raw image
[eulerinref]Euler angles should be taken from <references> not common-lines file
[frac=<num>/<denom>]Operate only on a fraction of the raw images
[verbose[=<n>]]More verbose output
[matrix]Generates similarity matrix for entire data set in matrix.dat
[logit=<label>]This specifies a label to use in the particle.log file. This records particle number classification throughout a refinement.

Usage:

classesbycl start.hed proj.hed

Description

This classifies a set of raw particles based on a set of reference projections. The output is a set of cls*.img files, one file for each reference image. This program differs from classesbymra in the use of self common-lines maps for performing the actual comparisons.
