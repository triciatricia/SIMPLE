helixhunter

This program is used to locate possible helices in intermediate resolution maps.

helixhunter <input> <output> <a/pix> [cyl] [int=<file>] [da=<angle>] [keep=<sigmamult>] [filt=<res>]

Parameters:


<input>Input 3D map at 5-8A resolution
<output>Output 3d map of helix positions
<a/pix>Angstroms/pixel in 3D map. Must be <2.
[cyl]Also outputs a 3d map containing the cylinder used in the search.
[int=<file>]Outputs the raw correlation files describing helix locations.
[da=<angle>]Angular step used in the hunt.
[keep=<sigmamult>]Specifies the peak strength that defines a helix.
[filt=<res>]Prefilter resolution in angstroms.

Usage:

helixhunter input.mrc output.mrc 1.5

Description

This is superseded by helixhunter2

Visit web page Helixhunter and Foldhunter for more information.
