cosec2iv

This program converts helices in COSEC secondary structure elements file format to Open Inventor format

cosec2iv <input COSEC file> <output IV file> [verbose]

Parameters:


<input COSEC file>The input COSEC format file
<output IV file>The output Open Inventor format file
[verbose]will output some intermediate information

Usage:

cosec2iv in.cosec out.iv verbose

Description

Visit web page Helixhunter and Foldhunter for more information.
