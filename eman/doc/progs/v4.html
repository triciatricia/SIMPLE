V4

 

An interactive 3d model, projection viewing program


Usage
v4 <3d model 1> <3d model 2> ...
 
<3d model> a list of files containing valid 3d models

Typical Usage
v4 threed.1a.mrc threed.2a.mrc

Description
This is a simple program designed to interactively rotate and compare 3d models side by side. This program generates projections rather than the conventional 3d isosurface renderings. For isosurfaces vis5d is the suggested program. This program allows you to see how visible differences between models are in the projections in any orientation. Euler angles are linked between all models.
