make3dicos2

Do not use this program directly. It is called from make3dicos.

make3dicos2 <input file> <outfile> [pad=<padsize>]

Parameters:


<input file>File containing class averages
<outfile>Output 3D MRC file
[pad=<padsize>]To reduce Fourier artifacts, the model is typically padded by ~25%

Usage:

make3dicos classes.1.img out=threed.1.mrc pad=128

Description

This program should only be called from make3dicos. It does the hard work of building a 3d icosahedral model in Fourier space. However, it uses VolFrag objects, which aren't very useful outside the context of these 2 programs
