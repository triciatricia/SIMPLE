fpweight

This program assigns weights to the various axes in a footprint file

fpweight <footprint file> [tfp]

Parameters:


<footprint file>Input footprint file to be weighted
[tfp]Writes a file which can be used to visualize the space

Usage:

fpweight proj.fp

Description

This program attempts to determine the degree of correlation between each axis (dimension) in the footprint file and the euler angles. It produces weights suitable for use by 'classes' or 'classesrefs'.

This program is experimental and not for general use.
