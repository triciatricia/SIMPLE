tiltseries

Aligns a set of images from a tilt series and assigns Eulers

tiltseries <input file> <output file> <dang> [indccl] [fixang=<ang>] [mask=<pixel radius>] [skip=<image0[,imageN]>] [maxshift=<pixels] [vertical]

Parameters:


<input file>Images to be processed
<output file>Processed output
<dang>Angular step between images
[indccl]Writes individual CCL images to ccl.hed
[fixang=<ang>]If the common-line orientation is known, specify it here
[mask=<pixel radius>]Mask type 4 (step cut off to zero), defaults to xSize()/3
[skip=<image0[,imageN]>]The image index in the input file to skip for alignment.
[maxshift=<pixels]Maximum shift in translational alignment, defaults to xSize()/4
[vertical]Align images so that tilt axis is vertical.

Usage:

tiltseries particle.hed particle.tilt.hed 10

Description

This program is still somewhat experimental

This proram is used for analysis of tilt-series, ie - tomography. This program will take a tilt series of a single particle, mutually align them, determine the tilt axis, and write the aligned images to a new file with Eulers.
