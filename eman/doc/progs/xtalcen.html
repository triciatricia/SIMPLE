xtalcen

This program will take a diffraction pattern, locate the center, remove the beamstop and do a background subtraction

xtalcen <input file [out=<output name>] [center=<x>,<y>] [filtrad=<rad>] [bgsub] [zerothr=<sigma mult>]

Parameters:


<input file<input file>
[out=<output name>]Optional output image
[center=<x>,<y>]No automatic centering
[filtrad=<rad>]Radius of background elimination filter for centering, should be larger than peak size, default 10
[bgsub]Do a background subtraction
[zerothr=<sigma mult>]Threshold for zeroed data as a sigma multiplier, default 1.5

Usage:

xtalcen edp1.mrc out=edp1a.mrc bgsub

Description

This program will find the center of a diffraction mode diffraction pattern, mask out the beamstop

and do an optional background subtraction.
