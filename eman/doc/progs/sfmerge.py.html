sfmerge.py

Combines a structure factor from CTFIT with a simulated curve for use in CTF correction

sfmerge.py <ctfit file> <high res file> <cutoff S> <output>

Parameters:


<ctfit file>File produced by CTFIT
<high res file>2 column text file containing high resolution structure factor
<cutoff S>Spatial frequency at which the transition should occur
<output>Output filespec

Usage:

sfmerge.py out.txt high.txt .04 sim.sf

Description

This program will combine a structure factor file generated by CTFIT with the 'save 1 column' command (save column 11), with a 2 column structure factor file at high resolution. This requires that the data sets have been fit so the calculated structure factor matches well at low spatial frequencies. If you wish to actually use this program, you should probably contact sludtke@bcm.tmc.edu first.

note the ctfit file contains amplitudes, the high res sf file contains intensities, and the output contains intensities
