matrixclass

This program will take a distance matrix and a pre-embedded space

matrixclass <input file> <reference file> <output file> [fixq=<val>] [classes]

Parameters:


<input file>Source matrix file from classesbymra
<reference file>Already embedded locations
<output file>Destination file (ascii)
[fixq=<val>]Output with a fixed quality factor rather than the calculated one
[classes]This will classify the particles and output a set of cls files like classesbymra.

Usage:

matrixclass matrix.dat groel.loc class.loc

Description

This program is still experimental.

This program is used in conjunction with matrixembed and the 'matrix' option of classesbymra. It is used to assign embedded locations to a set of points based on distances to a set of already embedded points. The typical sequence would be as follows:

  • classesbymra proj.hed proj.hed matrix
  • matrixembed matrix.dat proj.loc euler=proj.hed
  • classesbymra start.hed proj.hed matrix
  • matrixclass matrix.dat proj.loc start.loc
