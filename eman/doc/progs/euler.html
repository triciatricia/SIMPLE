euler

This program calculates a cross common lines image between 2 source images

euler <input file> <#1> <#2> <output file>

Parameters:


Description

This program will take 2 input images and calculate a square image with cross common-line dot products as the pixels in the image. This is an experimental utility used for program testing and design.
