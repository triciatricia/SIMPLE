c1startup

This program uses cross-common lines to generate a 3D model from a set of projections

c1startup <input file> <max [fixed=<int gt 0> [output=<file of numbers>

Parameters:


<input file>Source file with class averages
<max<max>
[fixed=<int gt 0>[fixed=<int gt 0>]
[output=<file of numbers>[output=<file of numbers>]

Usage:

c1startup good.img

Description

This program is used to generate and initial 3D model for refinement in cases where there is no symmetry, or the symmetry specific routines don't work very well. Generally you just specify the input file containing a variety of good class averages (usually 10-20). The model will be written to threed.0a.mrc.

This routine will work only on some models. It is very sensitive to correct particle centering and similarities between views in different orientations. If you have a model with symmetry or pseudosymmetry, one of the symmetry- specific routines should be used.
