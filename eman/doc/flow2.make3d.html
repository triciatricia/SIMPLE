Build 3D Model from Class Averages

Once the particles have been aligned and averaged into particles, make3d is run on the class averages to produce a 3D model. Since make3D operates in Fourier space, and produces some undesirable artifacts in real-space, a real-space iterative procedure is then used to clean up some of the artifacts.
