Particles

The reconstruction process begins with particle data. The particles are phase corrected if CTF correction is being performed, or filtered to the 1st zero of the CTF if no CTF correction is being performed. In both cases, the particles from each micrograph are examined with ctfit to either set the CTF parameters in the header and perform phase correction or determine the filter frequency.
