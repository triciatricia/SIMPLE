Classify by Projection

Once a reprojection of a model has been made, a set of particles may be classified with respect to  these reprojections.  Each particle image is compared with each reprojection (which therefore forms the original reference for each class). This comparison proceeds as follows: the CTF of the particle is applied to each reprojection. A quality factor for the match is then assigned between particle and appropriately modified reprojection. Each particle is associated with the projection it matched best.
